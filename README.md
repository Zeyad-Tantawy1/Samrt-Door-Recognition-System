# Samrt-Door-Recognition-System
A Python-based facial recognition system for secure access control. Uses OpenCV and deep learning to detect and recognize faces in real-time. Currently implements face detection, recognition, and live camera testing. Designed for easy integration with smart door mechanisms. Future plans include Telegram alerts and access logging.
